# Screencast Recording Guide for Meta App Review

## What to Record for whatsapp_business_messaging Permission

### Required Screen Recording Flow:

**1. Login & Dashboard (10-15 seconds)**
- Show the login screen of ProDigiChat
- Navigate to the main dashboard
- Show the WhatsApp/WhatsApp settings section

**2. WhatsApp Connection (20-30 seconds)**
- Click "Connect WhatsApp" button
- Show the Meta OAuth popup
- Demonstrate user logging into Facebook/Meta
- Show permission request screen
- Show successful connection confirmation

**3. Contact Management (15-20 seconds)**
- Navigate to contacts section
- Show how to import contacts (with consent notice)
- Display contact list management
- Show opt-in/consent management features

**4. Template Creation (20-25 seconds)**
- Go to message templates section
- Create a new message template
- Fill in template details (name, category, content)
- Submit template for WhatsApp approval
- Show template status

**5. Message Sending (20-30 seconds)**
- Select a contact from the list
- Choose to send a message
- Select or create a message template
- Send the message
- Show message sent confirmation

**6. Analytics & Tracking (15-20 seconds)**
- Navigate to analytics dashboard
- Show message delivery reports
- Display open rates and engagement metrics
- Show real-time message status

**7. Settings & Configuration (10-15 seconds)**
- Show WhatsApp settings page
- Display connected business account
- Show webhook configuration
- End with business profile overview

### Total Recording Time: 2-3 minutes

### Recording Tips:
- Use clear screen resolution (1080p recommended)
- Speak clearly explaining each step
- Show mouse cursor movements
- Highlight important buttons/sections
- Ensure all text is readable
- Test audio quality if adding narration

### Key Features to Highlight:
- **OAuth Integration**: Show secure Meta authentication
- **Consent Management**: Demonstrate opt-in compliance
- **Template System**: Show WhatsApp template approval process
- **Message Sending**: Display actual message functionality
- **Analytics**: Show tracking and reporting capabilities
- **Security**: Show business account management

### Before Recording:
- ✅ Test all features work properly
- ✅ Have test data ready (contacts, templates)
- ✅ Ensure WhatsApp Business account is connected
- ✅ Test screen recording software
- ✅ Prepare script for narration (optional)

### After Recording:
- ✅ Review video quality and clarity
- ✅ Ensure all required features are shown
- ✅ Check video length (keep under 5 minutes)
- ✅ Verify audio is clear (if narrated)
- ✅ Save in compatible format (MP4 recommended)

### Upload Requirements:
- File size: Under 100MB
- Format: MP4, MOV, or AVI
- Resolution: Minimum 720p, recommended 1080p
- Duration: 2-5 minutes maximum
- Must show end-to-end user experience

### Common Mistakes to Avoid:
- ❌ Recording too fast (slow down for clarity)
- ❌ Skipping important steps
- ❌ Not showing actual functionality
- ❌ Poor video quality
- ❌ Missing consent/opt-in features
- ❌ Not showing template approval process
